# Neon Snake Game

A sophisticated, mobile-friendly Snake game built with HTML5 Canvas.

## Features
- **Neon Visuals**: Glowing effects and smooth animations.
- **Mobile Optimized**: Swipe controls and responsive layout.
- **iOS Ready**: Add to Home Screen for a native app-like experience.
- **High Score**: Saves your best score locally.

## How to Run
1. Open `index.html` in any modern web browser.
2. For the best experience on iOS:
   - Host the files on a web server (or use a service like GitHub Pages).
   - Open the URL in Safari on your iPhone/iPad.
   - Tap the "Share" button.
   - Select "Add to Home Screen".
   - Launch the game from your Home Screen to play in fullscreen mode.

## Controls
- **Desktop**: Arrow keys to move, Space to start/restart.
- **Mobile**: Swipe to change direction, Tap to start/restart.
